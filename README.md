### Фичи:

- Раскладка Jiran (ну, почти...)
- Backlight
- Двойной shift работает как Enter
- Отображение состояния Caps/Num lock на экране
- отображение на экране текущего слоя клавы
- Отображение на экране активного на компе языка ввода (нужен агент на компе)
- Отображение активного на компе языка ввода в виде горящей/погасшей подсветки (нужен агент на компе)
- Активация режима Bootloader:
  - Если сбросить при нажатой кнопке 3 (только для левой половины)
  - Если на 3-м слое нажать 3 (на левой половине) или 8 (на правой)
  - Если сбросить при нажатой кнопке Boot на макетке (стандарт для RP2040)
  - "двойной клик" по сбросу
- На 3-м управляющем слое кнопка 5 (6) выдаёт на экран логотип и выплёвывает кой-какую информацию в debug console

### Логотип

- Подготовить файл `logo/logo.png` с разрешением 320x1280 пикселей (вертикальная ориентация). Лучше всего будет смотреться белые буквы на черном фоне.
- В скрипте `logo/create-logo-animation.py` подобрать под себя количество и длительность кадров анимации.
- Если логотип скорее черный на белом, придётся подправить скрипт:
  - `if img_1bit.getpixel((x, y)) > 0:` заменить на `if img_1bit.getpixel((x, y)) == 0:`
  - `current_frame.putpixel(px, 255)` заменить на `current_frame.putpixel(px, 0)`
- запустить питонский скрипт:

```bash
cd logo
python3 create-logo-animation.py
```

- появится папка с отдельными кадрами, а также анимированный файл `logo.gif`
- натравить на него преобразователь картинок:

```bash
rm -rf converted-image && mkdir -p converted-image && qmk painter-convert-graphics -f mono2 -o ./converted-image -i logo.gif
```

- содержимое папки `logo/converted-image/logo.qgf.*` переписать в `split/display/` с заменой файлов.
- Пересобрать прошивку.
